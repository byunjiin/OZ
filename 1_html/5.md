# _HTML문서기본구조_

### _html_

`<head>+<body>`

```js
<!DOCTYPE html> //문서 타입
<html>
  <head>
  문서의 정보
  </head>
  <body>
  화면에 표시될 내용
  </body>
</html>
```

#### _lang특성_

`html을 지정된 언어로 작성`

- html lang="en" : 영어(기본값)
- html lang="ko" : 한국어(주로 사용)

### _head_

`문서의 정보`

#### _meta태그_

head에 적고, `charset의 타입을 알려줌`. 문서의 인코딩 방식을 확인할 수 있음.

- meta charset = "utf-8"

#### _title태그_

head에 적고, `문서의 제목을 뜻함`. 탭에 표시.

### _body_

`화면에 표시될 내용`
