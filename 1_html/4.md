# _태그(Tag)와 속성(attributes)_


### _태그_

`<>`

#### _태그의 구성요소_

- 태그명
- 시작태그
- 끝 태그
- 콘텐츠

#### _태그 작성시 주의 사항_

- 태그 이름은 소문자로 쓴다
- 시작 태그와 끝 태그는 정확히 입력한다
- 태그의 콘텐츠로 또 다른 태그가 포함될 수 있다
- 들여쓰기를 적절하게 사용하는것이 좋다

### _속성_

`태그의 부가적인 기능을 정의하는 것`

#### _속성의 특징_

- 시작 태그의 내부에 정의
- 개수 제한은 따로 없음
- 태그에 추가할 속성이 여러 개인 경우 각각을 구분하기 위해 공백 사용
